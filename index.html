checkpoint 1

Section 1 : Sensibilisation générale

Bases du développement Web :

Qu’est-ce que le développement Web et pourquoi est-il essentiel dans le contexte d’Internet ?

reponse: Le développement web est le processus de création de stie et d'application web.
Dans le contexte d'internet, le developpement web trouve son importance en ce sens où
il offre une présence en ligne à travers les sites web et applications disponible avec internet.

Connaissance HTML :

Avez-vous entendu parler du HTML ? Si oui, que signifie HTML et quel est son objectif principal dans le développement Web ?

reponse: HyperText Markup Language (HTML) est un langage de balisage qui sert à structurer des pages web.
Il peut etre perçu comme le squelette d'une page web.

Section 2 : Familiarité avec CSS

Compréhension CSS :

Connaissez-vous CSS ? Si oui, expliquez brièvement à quoi sert CSS dans le développement Web.

reponse: Cascading StyleSheet (CSS) est utlisé pour styliser(décorer) une page web.

Section 3 : JavaScript et son importance

Introduction à Javascript :

Avez-vous entendu parler de JavaScript ?
Si oui, qu’est-ce que JavaScript et pourquoi est-il crucial pour créer des sites Web interactifs ?

reponse: Javascript est un langage de programmation puissaont et flexible, utlisé en developpement web.
il alimente le comportement dynamique de la plupart des sites web.

Section 4 : Introduction à React et Node.js

Sensibilisation à React et Node.js :

Connaissez-vous React et Node.js ? Si oui, fournissez une brève description
de ce pour quoi React et Node.js sont couramment utilisés dans le développement Web.

Reponse: je ne les connais que de nom.

Section 5 : Question bonus

Intentions d'apprentissage :

Pourquoi souhaitez-vous apprendre le développement Web et qu'espérez-vous réaliser
en acquérant des compétences en HTML, CSS, JavaScript, React et Node.js ?

Reponse: L'informatique est pour moi un domaine fascinant.
Ça l'est devenu encore après ma découverte du développement web.
Grande fut ma fascination face au mécanisme derrière la création de site web et d'application.
De plus, au vue de la place qu'occupe l'informatique en général,
il est plus que nécessaire de s'y connaitre ne serait-ce qu'un tout petit peu.
c'est ainsi que commence mes premier pas vers le développement web.
En acquérant ces compétences, je souhaite etre en mésure de créer tout types de site de A à Z.
